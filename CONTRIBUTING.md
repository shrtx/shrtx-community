# Contributing to SHRTX Community

Welcome to the SHRTX community space.

This repository is used for **community discussions and feedback about SHRTX tools**.

## What this repository is for

Use this space to:

• Ask questions about SHRTX tools  
• Report bugs or unexpected behavior  
• Suggest new tools or improvements  
• Share workflows or use cases  
• Discuss ideas related to the platform

Start a discussion here:

https://github.com/shrtx/shrtx-community/discussions

## Code Contributions

SHRTX platform development happens internally.

If you want to contribute ideas or report issues, please open a discussion in this repository.

Community feedback helps guide future tool development.

## Community Guidelines

Please be respectful and constructive when participating in discussions.

Spam, abuse, or malicious content may be removed.
